# 対象
1. createTask
2. completeTask
3. reopenTask

# テストケース
1. createTask
   - | 入力 | 検証すること |
     | :--- | :--- |
     |"new Task"|1. kind: "ok"<br>2. idが設定されていること <br> 3. タイトル: "new Task"<br>4. ステータス: "open"|
     |"n"|1. kind: "ok"<br>2. idが設定されていること <br>3. タイトル: "n"<br>4. ステータス: "open"|
     |空文字| 1.kind: "error"<br>2. InvalidTaskTitleErrorのインスタンスであること|
     |半角スペースだけ| 1.kind: "error"<br>2. InvalidTaskTitleErrorのインスタンスであること|
     |全角スペースだけ| 1.kind: "error"<br>2. InvalidTaskTitleErrorのインスタンスであること|

2. completeTask
   - | 入力 | 検証すること |
     | :--- | :--- |
     |生成済みのタスクでかつステータスが`open`|1. kind: "ok"<br>2. ステータス: "completed"<br> 3. id、タイトルが変わっていないこと|
     |生成済みのタスクでかつステータスが`completed`|1. kind: "error"<br>2. TaskAlreadyCompletedErrorのインスタンスであること|

3. reopenTask
   - | 入力 | 検証すること |
     | :--- | :--- |
     |生成済みのタスクでかつステータスが`completed`|1. kind: "ok"<br>2. ステータス: "open"<br> 3. id、タイトルが変わっていないこと|
     |生成済みのタスクでかつステータスが`open`|1. kind: "error"<br>2. TaskAlreadyOpenErrorのインスタンスであること|
